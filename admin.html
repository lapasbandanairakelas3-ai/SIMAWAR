URL GAS untuk menyimpan data absensi ke Google Sheets

Panduan Setup
📊 Google Apps Script
  1. Buat Google Sheets baru
  2. Buka Extensions → Apps Script
  3. Paste kode GAS dari dokumentasi
  4. Deploy sebagai Web App (Anyone)
  5. Salin URL dan paste di atas
🗄️ Supabase Tables Needed
  • site_config (id, site_name, site_desc, logo_url, favicon_url, gas_url)
  • pegawai (id, user_id, nama, jabatan, pangkat, username, email, role, status)
  • wbp (id, nama, foto_url, jk, no_registrasi, blok_id, tgl_masuk, masa_pidana, kasus, status)
  • blok (id, nama, kapasitas, jk, keterangan)
  • absen_session (id, pegawai_id, blok_id, tgl, waktu_mulai, waktu_selesai, status)
⚠️ Penting

Pastikan RLS (Row Level Security) Supabase dikonfigurasi dengan benar. Lihat file supabase.sql untuk schema lengkap.
